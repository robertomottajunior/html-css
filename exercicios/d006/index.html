TAGs em HTML

Em HTML5, podemos inserir os elementos da lista à seguir. Passe o mouse sobre os termos para descobrir qual TAG deverá usar em cada caso.

  • Parágrafos
  • Título (cabeçalho)
  • Imagens
  • Abreviações ("legendas")
  • Códigos-fonte
  • Listas ordenadas
  • Listas não ordenadas
  • Linhas dentro de uma lista
  • Quebra de linha
  • Hyperlinks
  • Títulos
  • Sub-títulos
  • Citações (quotes)
  • Textos pré-formatados
  • Formato de endereço
  • Favicon (link:favicon... Fica no Head)
